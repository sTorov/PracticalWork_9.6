# PracticalWork_9.6

## Task 1  
1. Создайте свой тип исключения.  
2. Сделайте массив из пяти различных видов исключений, включая собственный тип исключения.  
3. Реализуйте конструкцию `TryCatchFinally`, в которой будет итерация на каждый тип исключения (блок `finally` по желанию).
В блоке `catch` выведите в консольном сообщении текст исключения.
___
***Описание проекта:***  

В данном проекте создан собственный вид исключения `OneOrTwoException`, предназначеный для проверки значения: либо 1, либо 2.
Если значение отлично от 1 или 2 - происходит исключение.
Также создан список из 5-и исключений. При помощи цикла и конструкции `TryCatch` выведены сообщения всех исключений из списка на консоль. 

___
## Task 2
Создайте консольное приложение, в котором будет происходить сортировка списка фамилий из пяти человек. Сортировка должна происходить при помощи события. Сортировка происходит при введении пользователем либо числа 1 (сортировка А-Я), либо числа 2 (сортировка Я-А).

Дополнительно реализуйте проверку введённых данных от пользователя через конструкцию `TryCatchFinally` с использованием собственного типа исключения.
___
***Описание проекта:***

В данном проекте происходит сортировка списка из 5-и человек. Сортировка производится по первой букве фамилии человека при помощи события.  

Также реализована возможность
выбора сортировки:  
 - по возрастанию (А-Я)  
 - по убыванию (Я-А).
